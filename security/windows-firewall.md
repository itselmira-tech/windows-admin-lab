# Windows Firewall

## Objective

Review the Windows Firewall configuration and verify that network protection is enabled.

---

## Windows Security

The Firewall & network protection section allows administrators to manage firewall settings for different network profiles.

Profiles:

- Domain
- Private
- Public

---

## PowerShell Command

```powershell
Get-NetFirewallProfile
```

This command displays the status of Windows Firewall profiles.
