# Microsoft Defender

## Objective

Review Microsoft Defender protection status using the Windows Security application and PowerShell.

---

## Windows Security

Microsoft Defender is the built-in antivirus solution in Windows. It provides:

- Real-time protection
- Virus and threat protection
- Protection history
- Security intelligence updates

---

## PowerShell Command

```powershell
Get-MpComputerStatus
```

This command displays the current Microsoft Defender status, including real-time protection, antivirus status, signature versions, and scan information.
