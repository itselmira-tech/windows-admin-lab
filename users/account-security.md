# Account Security

## Objective

Review Windows account security settings and user privileges.

---

## Password Policy

Command:

```powershell
net accounts
```

The command displays local password policy settings.

---

## Current User Information

Command:

```powershell
whoami /user
```

Displays the current user's identity information.

---

## User Group Membership

Command:

```powershell
whoami /groups
```

Displays groups assigned to the current user account.
