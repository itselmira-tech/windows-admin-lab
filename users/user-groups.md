# User Groups

## Objective

Explore local Windows user groups and their members.

---

## List Local Groups

Command:

```powershell
Get-LocalGroup
```

This command displays all local groups available on the Windows system.

---

## Administrators Group

Command:

```powershell
Get-LocalGroupMember Administrators
```

This command shows users who have administrator privileges.
