# Basic PowerShell Commands

## Objective

Practice common PowerShell commands used in Windows administration.

---

## Commands

### Get-ComputerInfo

Displays detailed information about the operating system and hardware.

### $PSVersionTable

Displays the installed PowerShell version.

### Get-Process

Lists running processes.

### Get-Service

Lists Windows services.

### Get-NetAdapter

Displays installed network adapters.

- Process Monitoring
