# Disk Cleanup

## Objective

Review temporary files and available disk space that can be safely cleaned.

---

## Tool

Disk Cleanup (cleanmgr.exe)

---

## Purpose

Disk Cleanup helps remove unnecessary files such as:

- Temporary files
- Recycle Bin contents
- Windows Update cache
- Delivery Optimization files
