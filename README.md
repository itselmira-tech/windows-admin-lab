# Windows Administration Lab

## Overview

This repository documents hands-on Windows administration tasks performed on a Windows 11 system. The project focuses on user management, security, system maintenance, Windows services, and basic PowerShell administration.

The goal is to build practical IT support and system administration skills through real-world exercises and documented configurations.

---

## Objectives

- Learn Windows administration fundamentals
- Manage local users and groups
- Configure Windows security features
- Explore Windows services and startup programs
- Practice basic PowerShell commands
- Perform system maintenance tasks
- Document findings with screenshots and reports
