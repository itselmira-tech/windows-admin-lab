This repository documents practical Windows administration tasks performed on a Windows 11 system.

The project focuses on real-world IT support activities including:

- User Management
- Windows Services
- Microsoft Defender
- Windows Firewall
- Windows Update
- PowerShell Administration
- System Maintenance
- Basic Troubleshooting
